# Create Username Wordlists
- Using [Username Anarchy](https://github.com/urbanadventurer/username-anarchy.git).
	``` bash
	./username-anarchy john doe > john_doe_usernames.txt
	```
- Using [CUPP](https://github.com/Mebus/cupp).
	```
	cupp -i
	```
